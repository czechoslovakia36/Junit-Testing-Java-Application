#Junit test lifecycle 
<ul>
<li>@BeforeAll</li>
<li>@BeforeEach</li>
<li>@Test</li>
<li>@AfterEach</li>
<li>@AfterAll</li>
</ul>

<hr>


<p>@BeforeAll,@BeforeEach -used to perform initialization tasks for tests</p>
<p>@AfterAll , @AfterEach - used to perform cleanup tasks for tests</p>
